# OnlineShop
## 一款模拟电商小程序
### 具有12大项目模块
1.商品首页
2.商品列表
3.购物车
4.授权页面
5.商品搜索
6.商品收藏
7.商品分类
8.商品详请
9.结算页面
10.订单列表
11.个人中心
12.意见反馈
